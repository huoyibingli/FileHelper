ObsConfig 属性

ObsConfig 类型公开以下成员。

属性
  名称说明
AsyncSocketTimeout
异步请求的超时时间，单位：毫秒。
AuthType
连接OBS使用的鉴权方式，当开启协议协商时，该值被忽略。
AuthTypeNegotiation
是否进行鉴权方式协商，默认值：true。
BufferSize
上传对象时的读写缓存大小。
ConnectionLimit
允许打开的最大HTTP连接数。 默认值为1000。
Endpoint
连接OBS的服务地址。
KeepAlive
是否使用长连接，默认值：true。
MaxErrorRetry
请求失败后最大的重试次数，默认值为3。
MaxIdleTime
连接池中连接的最大空闲时间，单位：毫秒。 默认值为30000毫秒。
PathStyle 已过时。
判断是否采用路径访问方式，true使用路径访问方式，false使用虚拟主机访问方式，默认值：false。 注意：如果设置了路径方式，无法使用OBS 3.0版本桶的新特性。
ProxyDomain
代理服务器的域。
ProxyHost
代理地址。
ProxyPassword
连接代理服务器时使用的密码。
ProxyPort
代理端口。
ProxyUserName
连接代理服务器时使用的用户名。
ReadWriteTimeout
读写数据超时时间，单位：毫秒。 默认值为60000毫秒。
ReceiveBufferSize
套接字接收缓冲区的大小。
SecurityProtocolType
HTTPS协议类型。
Timeout
请求超时前等待的时间，单位：毫秒。
ValidateCertificate
是否验证证书。
