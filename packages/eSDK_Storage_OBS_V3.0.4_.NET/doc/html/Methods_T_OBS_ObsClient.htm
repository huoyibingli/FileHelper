CreateBucket
创建桶。 新创建桶的桶名在OBS中必须是唯一的。 如果是同一个用户重复创建同一区域的同名桶时返回HTTP状态码200。 除此以外的其他场景重复创建同名桶返回HTTP状态码409，表明桶已存在。一个用户可以拥有的桶的数量不能超过100个。
